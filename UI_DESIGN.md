# AI小说创作工坊 - UI设计方案 v3.0

## 设计目标

创建美观、易用、专业的桌面小说创作界面，提升用户体验和创作效率。

---

## 设计原则

### 1. 信息层次清晰
- 主要操作（自动创作、续写）突出显示
- 次要操作（设置、帮助）折叠收纳
- 信息密度适中，避免拥挤

### 2. 角色可视化
- 角色卡片式展示，一目了然
- 属性/武器/技能可视化
- 角色头像/首字母标识

### 3. 操作便捷
- 快速操作工具栏
- 右键快捷菜单
- 键盘快捷键支持

### 4. 视觉统一
- 统一的深色主题
- 一致的间距和圆角
- 清晰的图标系统

---

## 界面布局

### 三栏布局
```
┌──────────────────────────────────────────────────────┐
│ 菜单栏: 文件 | 创作 | 设置 | 帮助                      │
├──────────┬──────────────────────┬────────────────────┤
│ 左侧栏    │ 中央编辑区            │ 右侧面板            │
│ (240px)  │ (flex)               │ (280px)            │
│          │                      │                    │
│ 小说信息  │ 章节标题 + 导航        │ 智能体流程          │
│ 快速操作  │                      │                    │
│ 角色列表  │ 写作区域              │ 运行日志            │
│ 大纲列表  │                      │                    │
└──────────┴──────────────────────┴────────────────────┘
```

## 响应式断点

| 设备 | 宽度 | 布局调整 |
|------|------|----------|
| 大屏 | ≥1400px | 三栏完整显示 |

| 小屏 | <1024px | 侧边栏抽屉式 |

## 交互规范

### 状态指示
- 默认状态：基础样式
- 悬停状态：边框变亮，轻微上移
- 激活状态：背景变深
- 禁用状态：降低透明度

### 动画过渡
```css
transition: all 150ms ease;
```

### 滚动行为
- 平滑滚动
- 自定义滚动条样式
- 滚动条宽度：6px
